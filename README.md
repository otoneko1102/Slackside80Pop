# Slackside80Pop

# 『Slackside-Regular』について
[SlacksideOne](https://github.com/ManiackersDesign/slackside) に不足している漢字などのデータを [HachiMaruPop](https://github.com/noriokanisawa/HachiMaruPop) で補ったフォントです。
名前はそれぞれのフォント名由来です。
